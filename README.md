# Mare YT Downloader

A modern desktop GUI for **yt-dlp** built with **Python** and **CustomTkinter**.

---

## Features

- Analyze videos
- Download videos
- Download audio
- Thumbnail preview
- Clipboard paste
- Background downloading
- Progress bar
- Dark theme

---

## Planned Features

- Resolution selector
- Playlist support
- Download queue
- Download history
- Automatic updates
- Cookies support
- SponsorBlock
- Subtitle download
- Modern UI redesign

---

## Technologies

- Python 3.13
- CustomTkinter
- yt-dlp
- Pillow

---

## Installation

```bash
git clone https://github.com/MareG76/mare-yt-downloader.git

cd mare-yt-downloader

pip install -r requirements.txt

python main.py
```
